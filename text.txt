Ознакомимся с API.
   По ссылке https://reqres.in можно найти API для изучения и тренировки запросов к серверу.
   Совместно разберем GET запрос для вывода списка юзеров https://reqres.in/api/users.
   Попробовать получить данные с XMLHttpRequest и fetch. Необходимо разобраться с передачей параметров через GET запрос и получением данных.

Выведем данные в таблицу.
   После успешного получения данных выведем всех пользователей в таблицу.
   Добавим постраничную навигацию.
   И возможность настраивать количество отображаемых пользователей с помощью <select>.
   После любого изменения “настроек” таблицы (переход на другую страницу или настройка вывода) настройки должны сохранятся в localStorage.
